Geeklog Media Gallery Plugin Documentation

The following information explains the configuration options, autotags and templates of the Geeklog Media Gallery Plugin. For install or upgrade instructions please read the readme.html file found in the /path/to/geeklog/plugins/mediagallery/ directory.
