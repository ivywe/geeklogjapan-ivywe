概要

自由に属性（項目）を追加できるデータ管理プラグインです。

機能

  • データ毎にfeaturedimageの登録が可能：追加項目で、最初に表示される画像（ファイル保存）タイプの項目
  • 画像(featuredimage)付カテゴリ別フィード発行
  • ogpメタタグ編集（個別データ表示、カテゴリ別件数一覧、カテゴリ別一覧）
  • カスタム設定：各サイトで独自の言語ファイル、テンプレートが使用可能、テーマ変数の追加が可能custom参照
  • カテゴリ別にテンプレートの指定が可能

オプション機能

  • XML：XMLファイルのインポート・エクスポート
  • CSV：CSVファイルのインポート・エクスポート インポートスケジュール実行（疑似CRON）
  • MAPS:データをMapsプラグインのマーカーへエクスポート　エクスポートスケジュール実行（疑似CRON）

関連プラグイン(2015/10/07 update)

  • Mapsプラグイン
    Mapsプラグインのロード順はDataBoxより上位にしてください。
  • Assistfbプラグイン：Assistfbプラグインが有効な場合のみ 特別データコードが機能します。
    @fb_pageid:アクセスされたフェースブックページのデータ。
    データが未登録で新規作成権限があれば登録後、mydata更新画面へ遷移します
  • XMLSitemapプラグインのコンフィギュレーション設定のサイトマップの内容に databox を登録すると sitemap.xmlに出力します

制限事項(2015/10/07 update)

  1. Gl2.1未満のGeeklogではインストールできません。
  2. MySQL 5.0 未満ではインストールできません。
  3. Geeklogインストール時に「UTF-8を使用する」をチェックしたサイトのみ対応しています。
  4. MySQLのみ対応しています。
  5. 主にXrea仕様 および SaaSes サーバで動作作検証しています。

注意

同一サイトにUserBoxがある場合、バージョン（幹番）は同じである必要があります。
