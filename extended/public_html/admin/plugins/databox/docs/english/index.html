概要

自由にAdditional fieldを追加できるData 管理Plugin です.

機能

  • Data 毎にfeaturedimageのregist がable：Additional field で、At the first time display されるImages （File save ）Type 項目
  • Images (featuredimage)付Category 別feed 発行
  • ogpMeta tag 編集（Data display 、Category 別Number list 、Category 別list ）
  • custom Setting：各site で独自のlanguage File 、Template がuse able
  • Category 別にTemplate のSelect がable

関連Plugin

  • jQueryPlugin Version1.3以上：jQeryPlugin が有効な case にonly date pickerが動作します.
    jQueryPlugin のロード順はDataBoxより上位にしてください.    Gl2.0未満ではdate pickerは動作しません
  • MapsPlugin
  • AssistfbPlugin ：AssistfbPlugin が有効な case only 特別Data code が機能します.
    @fb_pageid:access されたフェースブックページのData .
    Data が未regist で新規作成access rights があればregist 後、mydataupdate Display へ遷移します
  • XMLSitemapPlugin のConfiguration Settingのsite マップの内容に databox をregist すると sitemap.xmlに出力します

制限事項

  1. Gl1.8未満のGeeklogではInstall できません.
  2. MySQL 5.0 未満ではInstall できません.
  3. GeeklogInstall 時に「UTF-8using」をcheck したsite only 対応しています.
  4. MySQLonly 対応しています.
  5. 主にXrea仕様 および SaaSes サーバで動作作検証しています.

注意

同一site にUserBoxがある case 、Version（幹番）は同じである必要があります.
