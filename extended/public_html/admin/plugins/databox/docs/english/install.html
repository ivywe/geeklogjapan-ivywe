DataBox Plugin

(2012/02/17 update)

Install

  1. DatabaseおよびFile のbackupをとります.
  2. Standard的なPlugin のFile 配置に従い、File をuploadします.
    1. plugins/　under に　databoxunder をupload
    2. admin/plugins/databox/under に　adminunder のFile をmove
    3. public_html/databox/under に　public_htmlunder のFile をmove
  3. Rootuser としてLogged in し，Plugin admin's Display でInstall を実行します

その他必要なDirectory ＆File

  • public_html/images/databox (Additional field のImages （外記憶）の記憶path)
  • public_html/databox_data (File save path)
  • private/data/databox/ (File save path)

    • File save pathは、Configuration Settingで変更ableです

Install 時に作成されるaccess rights

  1. databox.admin:DataBoxPlugin に関するすべての処理ができる.
  2. databox.edit:(user にupdate をpermit しないSettingになっていても)update ができる.
  3. databox.submit:(user に creat new をpermit しないSettingになっていても) creat new ができる.

Install 時に作成されるGroup

  1. DataBox Admin :databox.admin databox.edit databox.submit access rights have
  2. DataBox Editor :databox.edit　access rights have
  3. DataBox Submitters :databox.edit databox.submit　access rights have

実行時に作成されるFile

  • data/databoxconfig_bak.php　　Configuration のbackup

Uninstall procedure

  • DatabaseおよびFile のbackupをとります.
  • Rootuser としてLogged in し，Plugin admin's Display でdelete を実行します.
  • uploadしたFile をdelete します.

Uninstall すると Database上のDataBox専用Data は、すべてdelete されます.

通常のupgrade procedure

  1. DatabaseおよびFile のbackupをとります.
  2. Standard的なPlugin のFile 配置に従い、File をuploadします.
    1. plugins/　under に　databoxunder をupload
    2. admin/plugins/databox/under に　adminunder のFile をmove
    3. public_html/databox/under に　public_htmlunder のFile をmove
  3. Rootuser としてLogged in し，Plugin admin's Display でupdate を実行します

基本的なtesting version のupgrade procedure

Uninstall 、after delete, 、re Install してください.

Uninstall する前にConfiguration Settingのbackupを 作成しておくと、Install しなおした際にリストアすることができます.

Uninstall すると Database上の専用Data は、すべてdelete されます.

Uninstall すると uploadしたFile は、すべてdelete されます.

Table の追加変更が必要な case があります.

Group およびaccess rights の追加変更が必要な case があります.
