[databox:datalink ]

id: code:
linktext*2 		Data display link
例１：[databox:datalink id:1 ]
例２：[databox:datalink code:data1]
例３：[databox:datalink code:data1 Data ]
id とcode が共にSelect されている case はidを優先します
When linktext omitted, Data のTitle をlinktextとします.該当Data がない case はdisplay されません.
[databox:mydataentrylink ]
[databox:mydataentry ]

type_id
templatedir*1
linktext*2
 My data creat new link
例１：[databox:mydataentrylink ]
例２：[databox:mydataentrylink linktext]
例３：[databox:mydataentrylink type_id:1]
例４：[databox:mydataentrylink type_id:1 creat new ]
例５：[databox:mydataentrylink type_id:1 templatedir:XXX linktext]
When omitted type_id linktext, My dataの「 creat new 」link をdisplay.
When omitted type_id, creat new link をdisplay
When type_id selected and linktext omitted, Type name 称をlinktextとします. When type_id selected and no Type, not display.
When not have my data creat new access rights, not display.
