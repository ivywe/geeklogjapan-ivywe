Assist Plugin

Autotags(2013/01/28 update)

Format Description
[newstories:話題ID RSSFile ] 新着記事（おしらせ）
例1:[newstories:news news.rss]
例2:[newstories:]話題Select When omitted, はConfiguration Setting value、Setting valueがない case はALL
例3:[newstories:ALL]全話題
例4:[newstories:news|General] 話題の複数Select
[newstories2:話題ID RSSFile ほか例1参照] 新着記事（おしらせ）
例1:[newstories2:news rss_file:news.rss title_trim_length:50 intervalday:70 limitcnt:40 newmarkday:30 thtml:xxxx]
例2:話題news を全件display [newstories2:news intervalday:0]
例3:話題news を/Theme /xxxx/ のTemplate でdisplay [newstories2:news thtml:xxxx]
例4:[newstories2:ALL]全話題
例5:[newstories2:news|General] 話題の複数Select
[conf:Varialname ] $_CONF[Varialname ] の valueを返す
例1：[conf:site_url]
例2：[conf:site_name]
例3：[conf:site_mail]
例4：[conf:site_slogan]
[assist:usercount] regist user 数
[assist:usercount_location] locationregist user 数
[assist:staticpage_content] 例1:[assist:staticpage_content] id=blockpageの静的ページを無条件にdisplay
例2:[assist:staticpage_content id:xxxx urlkey:yyyy] urlor スクリプトの引数にyyyyを含む case にidがxxxx の静的ページをdisplay
例3:[assist:staticpage_content id:xxxx mode:home] ホームの case にidがxxxx の静的ページをdisplay
例4:[assist:staticpage_content id:xxxx mode:sub] ホーム以外の case にidがxxxx の静的ページをdisplay
例5:[assist:staticpage_content id:xxxx mode:except] Select したキーワードを含まないサブページの case にidがxxxx の静的ページをdisplay
Select された静的ページをdisplayaccess rights のないuser がaccess している case はdisplay されません
modeがSelect されている case 　urlkeyのSelect は無視されます
idがSelect not case は、blockpage がSelect されたものとします
multi language site の case 、id:xxxx_jpのlanguage idは、urlの第一引数or 現在のlanguage mode に基づき、自動的に変換されます.（第一引数優先）
[assist:user_homepage uid:9] Select user （When omitted, Logged in user ）のホームページ
[assist:user_location uid:9] Select user （When omitted, Logged in user ）の居住地
[assist:user_sig uid:9] Select user （When omitted, Logged in user ）の署name
[assist:user_photo uid:9 width:9 height:9] Select user （When omitted, Logged in user ）の写真
[assist:user_about uid:9 ] Select user （When omitted, Logged in user ）の自己紹介
[assist:user_pgpkey uid:9 ] Select user （When omitted, Logged in user ）のPGP鍵
[assist:uid] Logged in user のuid
[assist:username uid:9] Select user （When omitted, Logged in user ）のuser name
[assist:fullname uid:9] Select user （When omitted, Logged in user ）の氏name
[assist:showgroup] Logged in user の所属するGroup list 出力
[assit:newstories 話題ID RSSFile ほか例1参照] 新着記事（おしらせ）
例1:[assist:newstories]
例2:[assist:newstories topics:news rss_file:news.rss title_trim_length:50 intervalday:70 limitcnt:40 newmarkday:30 thtml:xxxx]
例3:話題news を全件display [assit:newstories topics:news intervalday:0]
例4:話題news を/Theme /xxxx/ のTemplate でdisplay [assist:newstories topics:news thtml:xxxx]
例5:[assist:ALL]　全話題
例6:[assist:newstories topics:news|General]　話題の複数Select
[assit:home_id] home or sub （language File で変更可）
[assit:login_status] member or guest （language File で変更可）
[assit:topic_id] 現在のDisplay の話題ID
[assit:sp_id] 現在のDisplay の静的ページID
[assit:rss RSSFile ほか例1参照] rss のdisplay
例1:[assist:rss urlrss:www.rentpro.co.jp/backend/databoxchintai.rss strip_tags:-1 description_trim_length:30 templatedir:aaa]
例2:[assist:rss]
[assist:btn typeほか例1参照 ] ソーシャルボタン・ボックス のdisplay
例1:[assist:btn type:fblikebtn templatedir:aaa]
例2:[assist:btn type:evclip]
例3:[assist:btn type:fbcomment]
例4:[assist:btn type:tweet]
例5:[assist:btn type:plusone]
[assist:truncate text:x maxlen:9 filler:x endchars:9] 文字列を区切る
例１；[assist:truncate text:あいうえおかきくけこ maxlen:3]
例２；[assist:truncate text:あいうえおかきくけこ maxlen:3 filler:...]
例３；[assist:truncate text:あいうえおかきくけこ maxlen:1 filler:... endchars:3]
[assist:cache id:top type:staticpages] キャッシュFile を返す
例１；[assist:cache id:top]
例２；[assist:cache id:top type:staticpages]
キャッシュFile は、キャッシュpath /type/id.html
キャッシュFile が存在しない case は、空白を返します
[assist:staticpage] 例１：[assist:staticpage id:xxxx]
例２：[assist:staticpage id:xxxx linktext]
linktext が省略されている case は、Title をlinktextとします.
linktext は、必ず最後に記述してください.半角スペースを含む内容も記述ableです.
該当ページがない case もlink はdisplay されます.
multi language site の case 、id:xxxx_jpのlanguage idは、urlの第一引数or 現在のlanguage mode に基づき、自動的に変換されます.（第一引数優先）
