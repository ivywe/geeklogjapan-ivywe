After save のDisplay 遷移 User's Display (aftersave) 作成した項目をdisplay(item) save した後、どのページにmove するかをSelect します.
  • Display 遷移なし(no): don't move.
  • 作成した項目をdisplay(item): 作成した記事をdisplay.
  • 項目list をdisplay(list): list をdisplay.
  • ホームをdisplay(home): site のHomeページ（Top page）をdisplay.
  • AdminTOPをdisplay(admin): Adminをdisplay.
  • Plugin トップをdisplay(plugin): Plugin トップをdisplay.
　notification: 現在対象Display はありません
