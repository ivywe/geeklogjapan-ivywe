Assist Plugin

Install

  1. DatabaseおよびFile のbackupをとります.
  2. Standard的なPlugin のFile 配置に従い、File をuploadします.
    1. plugins/　under に　assistunder をupload
    2. admin/plugins/assist/under に　adminunder のFile をmove
    3. public_html/assist/under に　public_htmlunder のFile をmove
  3. Rootuser としてLogged in し，Plugin admin's Display でInstall を実行します

その他必要なDirectory ＆File

  • public_html/assist_data (File save path)
  • private/data/assist/ (File save path)

    • 上記は、Configuration Settingで変更ableです

Install 時に作成されるGroup

  • Assist Admin assist.admin Has full access to Assist features

Install 時に作成されるFile

  • logs/assist_newsletter.log

    • File がすでに存在する case は、File 内容はdelete されます.

実行時に作成されるFile

  • data/assistconfig_bak.php　　Configuration のbackup
  • data/assist_reserv_(予約日時)　ニュースレターの予約File
  • log/assistconfig_bak.php　　Configuration のupdate 時のbackup

Uninstall procedure

  • DatabaseおよびFile のbackupをとります.
  • Rootuser としてLogged in し，Plugin admin's Display でdelete を実行します.
  • uploadしたFile をdelete します.

Uninstall すると Database上のAssist専用Data 、ニュースレターの予約は、すべてdelete されます.

通常のupgrade procedure

  1. DatabaseおよびFile のbackupをとります.
  2. Standard的なPlugin のFile 配置に従い、File をuploadします.
    1. plugins/　under に　assistunder をupload
    2. admin/plugins/assist/under に　adminunder のFile をmove
    3. public_html/assist/under に　public_htmlunder のFile をmove
  3. Rootuser としてLogged in し，Plugin admin's Display でupdate を実行します

基本的なtesting version のupgrade procedure

Uninstall 、after delete, 、re Install してください.

Uninstall すると Database上の専用Data は、すべてdelete されます.

Table の追加変更が必要な case があります.

Group およびaccess rights の追加変更が必要な case があります.

Adminのbackup＆リストアのDisplay でConfiguration 仕様を最新にupdate することができます.
