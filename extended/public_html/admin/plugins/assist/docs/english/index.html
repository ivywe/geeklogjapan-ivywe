Outline

Geeklogsite を種々補助するPlugin です

機能

  • user 関連：インポート（メール通知なし）delete （user ID範囲Select ）
  • 変数関連：擬似クーロン実行日を変更する
  • ニュースレター：記事をニュースレターとして送信.送信先を分けて送信予約も
    JapanizePlugin を先にロードしないと、Japanese メールが文字化けする case があります.
  • backup & restore ：assistPlugin のConfiguration のbackup & restore 初期化 update
  • いいねボタン等のAutotags（Theme variables ）
    facebookのいいねボタンusing case は、GeeklogのConfiguration SettingでFacebook OAuth Application IDのSettingが必要です.
  • Topページ、記事、話題のトップのMeta tag 編集（Theme variables ）
    GeeklogのConfiguration SettingのMeta tag を有効にするとDescription Meta tag がregist できます.

Related Plugins

  • Japanize : load earlier than Assist Plugin
  • merumagauser （wkymimguser）
  • Maps professional版only

制限事項

  1. 英語language File がない case のlanguage File の既定 valueはJapanese になっています.
  2. 文字code はutfonly 対応しています.
  3. MySQLonly 対応しています.
  4. 主にXrea仕様 および SaaSes サーバで動作作検証しています.

注意

Ver1.0　以前のVersionのassistPlugin が存在する case は、いったん Uninstall 、after delete, 、re Install してください.

Ver1.0　以前のVersionのAutotagsは、原則、そのままでもuse できます.
