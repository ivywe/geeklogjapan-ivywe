TinyMCEプラグインの共通設定

TinyMCEの設定

項目(変数) 初期値 説明
対象textarea(targets) 自動選択 TinyMCEを有効にするtextareaタグの種類を指定します。次の4つのうちのどれかです。
  • 「自動選択」: 記事/静的ページ/コメント/カレンダ/グループメール/ブロック編集時のtextareaタグでTinyMCEを有効にします。
  • 「すべてのtextareaタグ」: Geeklogで使用しているすべてのtextareaタグでTinyMCEを有効にします。
  • 「特定のtextareaタグ（CSSクラス名指定）」: Geeklogで使用しているすべてのtextareaタグのうち、「CSSクラス名」で指定したクラス名を持つものだけでTinyMCEを有効にします。
  • 「特定のtextareaタグ（CSS ID指定）」: Geeklogで使用しているすべてのtextareaタグのうち、「CSS ID」で指定したIDを持つものだけでTinyMCEを有効にします。
CSSクラス名(target_class) tinymce_enabled 「対象textarea」を「特定のtextareaタグ（CSSクラス名指定）」に設定した場合、この項目で指定したCSSクラス名を持つtextareaだけでTinyMCEが有効になります。
CSS ID(target_ids) array('introtext', 'bodytext', 'html_content') 「対象textarea」を「特定のtextareaタグ（CSS ID指定）」に設定した場合、この項目で指定したCSS IDを持つtextareaだけでTinyMCEが有効になります。
エディタの縦の長さ(height) 360 ツールバーを含むエディタの縦の長さをピクセルないしパーセントで指定します。ピクセルの場合は数字のみ、パーセントの場合はパーセントを数字の後につけます。この欄を空白にすると、TinyMCEが置き換わる前のtextareaタグの縦の長さと同じになります。
エディタの横の長さ(width) 90% エディタの横の長さをピクセルないしパーセントで指定します。ピクセルの場合は数字のみ、パーセントの場合はパーセントを数字の後につけます。この欄を空白にすると、TinyMCEが置き換わる前のtextareaタグの横の長さと同じになります。

TinyBrowserプラグインの設定

項目(変数) 初期値 説明
フォルダ作成時のパーミッション 777 アップロード時のフォルダ作成で必要となるパーミッションを8進数で指定します。先頭の0はつけないでください。Unix系サーバでのみ必要です。
ファイル名をチェックする はい アップロードされるファイルの名前をチェックするかどうかを指定します。
画像ファイルの拡張子 array('jpg', 'jpeg', 'gif', 'png') アップロードを許可する画像ファイルの拡張子を指定します。
メディアファイルの拡張子 array('swf', 'dcr', 'mov', 'qt', 'mpg', 'mp3', 'mp4', 'mpeg', 'avi', 'wmv', 'wm', 'asf', 'asx', 'wmx', 'wvx', 'rm', 'ra', 'ram') アップロードを許可するメディアファイルの拡張子を指定します。
アップロード禁止ファイルの拡張子 array('php', 'php3', 'php4', 'php5', 'phtml', 'asp', 'aspx', 'ascx', 'jsp', 'cfm', 'cfc', 'pl', 'bat', 'exe', 'dll', 'reg', 'cgi', 'sh', 'py', 'asa', 'asax', 'config', 'com', 'inc') アップロードを禁止するファイルの拡張子を指定します。
画像ファイル 0 アップロードできる画像ファイルのサイズ(単位：バイト)の上限を指定します。0で無制限になります。1MBなら、1048576を指定します。
メディアファイル 0 アップロードできるメディアファイルのサイズ(単位：バイト)の上限を指定します。0で無制限になります。1MBなら、1048576を指定します。
その他のファイル 0 アップロードできるその他のファイルのサイズ(単位：バイト)の上限を指定します。0で無制限になります。1MBなら、1048576を指定します。
画質 80 画質を1～99の範囲で指定します。この設定は、画像をリサイズしたり回転するときにしか使用されません。
サイズ(横) 0（リサイズなし） リサイズ後のサイズ（横）
サイズ（縦） 0（リサイズなし） リサイズ後のサイズ（縦）
サムネールのサイズ(px) 80 サムネール画像のサイズ（単位: ピクセル）
サムネールの画質 80 画質を1～99の範囲で指定します。この設定は、サムネール画像を作成するときにしか使用されません。
ポップアップウィンドウのサイズ(横) 770 TinyBrowserのポップアップウィンドウのサイズ(横)を指定します。
ポップアップウィンドウのサイズ(縦) 480 TinyBrowserのポップアップウィンドウのサイズ(縦)を指定します。
1ページあたりのアイテム数 0 ファイルを1ページあたりいくつ表示するかを指定します。0を指定すると、ページ分割せずにすべてを表示します。
タイムスタンプのフォーマット Y-m-d H:i タイムスタンプを表示する際に使用するPHPのdate関数のフォーマットを指定します。
