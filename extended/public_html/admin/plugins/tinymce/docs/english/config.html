Global settings for TinyMCE plugin

TinyMCE settings

Items (Variable) Default value Description
Target textarea(mode) Select automatically The kind of <textarea> tags to which you would like to attach TinyMCE. Valid values are:
  • Select automatically: <textarea> tags in Story/Staticpage/Comment/Calendar/Mail/Block editors
  • All <textarea> tags: All <textarea> tags used in Geeklog
  • Specific <textarea> tags(with a specific CSS class name): All <textarea> tags used in Geeklog which have a specific CSS class name
  • Specific <textarea> tags(with specific CSS IDs): All <textarea> tags used in Geeklog which have specific CSS IDs
CSS class name(target_class) tinymce_enabled The CSS class name of <textarea> tags to which you would like to attach TinyMCE when you choose "Specific <textarea> tags(with a specific CSS class name)" in the above option.
CSS IDs(target_ids) array('introtext', 'bodytext', 'html_content') The CSS class name of <textarea> tags to which you would like to attach TinyMCE when you choose "Specific <textarea> tags(with specific CSS IDs)" in the above option.
Editor height(height) 360 the height of TinyMCE (including toolbars) in pixels or percent. To specify in pixels, enter only a number. To specify in percent, enter a number followed by a '%'. If you leave this item blank, the height of the textarea tag TinyMCE replaces will be used.
Editor width(width) 90% the width of TinyMCE in pixels or percent. To specify in pixels, enter only a number. To specify in percent, enter a number followed by a '%'. If you leave this item empty, the width of the textarea tag TinyMCE replaces will be used.

TinyBrwoser plugin settings

Items (Variable) Default value Description
Folder permissions 777 Permissions needed in creating folders on Unix servers. Be sure to use octal numbers WITHOUT preceding zeros, for example, 777 or 755.
Clean filename Yes Whether to clean filenames on upload
Permitted image file extensions array('jpg', 'jpeg', 'gif', 'png') Extensions of image files you are allowed to upload
Permitted media file extensions array('swf', 'dcr', 'mov', 'qt', 'mpg', 'mp3', 'mp4', 'mpeg', 'avi', 'wmv', 'wm', 'asf', 'asx', 'wmx', 'wvx', 'rm', 'ra', 'ram') Extensions of media files you are allowed to upload
Prohibited file extensions array('php', 'php3', 'php4', 'php5', 'phtml', 'asp', 'aspx', 'ascx', 'jsp', 'cfm', 'cfc', 'pl', 'bat', 'exe', 'dll', 'reg', 'cgi', 'sh', 'py', 'asa', 'asax', 'config', 'com', 'inc') Extensions of files you are not allowed to upload
Image file maximum size 0 Maximum size (in bytes) of image files you are allowed to upload. 0 means "unlimited". If you want to specify 1MB (mega bytes), specify 1048576.
Media file maximum size 0 Maximum size (in bytes) of media files you are allowed to upload. 0 means "unlimited". If you want to specify 1MB (mega bytes), specify 1048576
Other file maximum size 0 Maximum size (in bytes) of other files you are allowed to upload. 0 means "unlimited". If you want to specify 1MB (mega bytes), specify 1048576
Image quality 80 Image quality (1 to 99, the higher the better). This setting will be used only when resizing or rotating images.
Width 0(= no resizing) The width of a resized image
Height 0(= no resizing) The height of a resized image
Image thumbnail size 80 The size of a thumbnail image (in pixels)
Thumbnail quality 80 Thumbnail quality (1 to 99, the higher the better). This setting will be used only when creating thumbnails.
Pop-up window width 770 The width of TinyBrowser pop-up window
Pop-up window height 480 The height of TinyBrowser pop-up window
File Pagination 0 The number of items per page. 0 means no paging.
Date format Y-m-d H:i Date format as per PHP date function
