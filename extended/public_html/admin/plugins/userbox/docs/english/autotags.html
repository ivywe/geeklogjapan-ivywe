UserBox Plugin

Autotags(2013/08/23 update)

書式 Description
[userbox:count Category idor Category code ] Data Number
例１：[userbox:count]
例２：[userbox:count category_id:1]
例３：[userbox:count category_code:japan]
例４：[userbox:count category_code:japan|america] Category code がjapan or america
例５：[userbox:count category_code:japan|groupA mode:and] Category code がjapan かつgroupA
[userbox:newlist　RSSFile ほか例２参照] 新着user
例１：[userbox:newlist]
例２：[userbox:newlist rss_file:userbox.rss title_trim_length:50 intervalday:70 limitcnt:40 newmarkday:30 templatedir:aaa category_id:99 category_code:x]
[userbox:profile id] user プロフィール
例１：[userbox:profile]
例２：[userbox:profile id:1 templatedir:aaa nohitmsg:yes]
[userbox:category　Category idor Category code 　ほか] Category 別list
例１：[userbox:category]
例２：[userbox:category category_id:1 templatedir:aaa nohitmsg:yes perpage:10 page:1 order:id expired:no]
例３：[userbox:category category_code:japan ]
例４：[userbox:category category_code:japan|america ] Category code がjapan or america
例５：[userbox:category category_code:japan|groupA mode:and] Category code がjapan かつgroupA
order にSelect できるのは、 random date orderno code title description id と　display 制限not Additional field のTheme variables.

perpage:(Number) Count of display items per page. if 0 display all items. (default: 0)
expired:no expired data はnot display.（ Default）
expired:yes expired data もdisplay.
[userbox:attributedatacount Theme variables value] Select AttributeがSelect valueのData のNumber
例１：[userbox:attributedatacount]
例２：[userbox:attributedatacount templatesetvar:pref value:27]
[userbox:lastmodified　theme directory　Additional field value ]
例１：[userbox:lastmodified]
例２：[userbox:lastmodified category_id:1]
例３：[userbox:lastmodified category_code:japan]
[userbox:lastcreated　theme directory　Additional field value ]
例１：[userbox:lastcreated]
例２：[userbox:lastcreated category_id:1]
例３：[userbox:lastcreated category_code:japan]
[userbox:attribute　theme directory　Additional field value 　ほか例２参照] Attribute別list
Select AttributeがSelect valueのData list
例１：[userbox:attribute field_id:99 value:xx ]
例２：[userbox:attribute field_id:99 value:xx templatedir:x nohitmsg:yes perpage:99 page:99 order:id]
例３：[userbox:attribute field_code:pref value:27 templatedir:x nohitmsg:yes perpage:99 page:99 order:id]

perpage:(Number) Count of display items per page. if 0 display all items. (default: 0)
[userbox:msg　Message ]
例１：[userbox:msg ] すべてのuser 用Message をdisplay
例２：[userbox:msg for:draftuser] draft user 用Message をdisplay
例３：[userbox:msg for:alluser] すべてのuser 用Message をdisplay
例４：[userbox:msg for:descriptionempty]Description未regist check をおこなう case 未regist のuser 用Message をdisplay
Message の内容は、language File に記述しています.
[userbox:categorycount　Category Group id or 　Category Group Code] Select Category Group のCategory Number
例１：[userbox:categorycount ] Paerent Group なしCategory のNumber
例２：[userbox:categorycount group_id:3]
例３：[userbox:categorycount group_code:pref]
Access rights のあるData がregist not Category のNumber not include
