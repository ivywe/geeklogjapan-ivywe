UserBox Plugin

(2012/02/17update)

Install

  1. DatabaseおよびFile のbackupをとります.
  2. Standard的なPlugin のFile 配置に従い、File をuploadします.
    1. plugins/　under に　userboxunder をupload
    2. admin/plugins/userbox/under に　adminunder のFile をmove
    3. public_html/userbox/under に　public_htmlunder のFile をmove
  3. Rootuser としてLogged in し，Plugin admin's Display でInstall を実行します

その他必要なDirectory ＆File

  • public_html/images/userbox (Additional field のImages （外記憶）の記憶path)
  • public_html/userbox_data (File save path)
  • private/data/userbox/ (File save path)

    • File save pathは、Configuration Settingで変更ableです

Install 時に作成されるaccess rights

  1. userbox.admin:UserBoxPlugin に関するすべての処理ができる.
  2. userbox.edit:(user にupdate をpermit しないSettingになっていても)update ができる.
  3. userbox.joingroup:(user にマイGroup のupdate をpermit しないSettingになっていても)update ができる.
  4. userbox.user:UserBoxにregist するuser .Logged in user をregist するSettingになっている case は、Logged in user はすべてregist されます.

Install 時に作成されるGroup

  1. UserBox Admin :userbox.admin userbox.edit userbox.submit access rights have
  2. UserBox Editor :userbox.edit　access rights have
  3. UserBox Manager :userbox.edit userbox.joingroup　userbox.user access rights have
  4. UserBox User :userbox.user　access rights have

実行時に作成されるFile

  • data/userboxconfig_bak.php　　Configuration のbackup

Uninstall procedure

  • DatabaseおよびFile のbackupをとります.
  • Rootuser としてLogged in し，Plugin admin's Display でdelete を実行します.
  • uploadしたFile をdelete します.

Uninstall すると Database上のUserBox専用Data は、すべてdelete されます.

通常のupgrade procedure

  1. DatabaseおよびFile のbackupをとります.
  2. Standard的なPlugin のFile 配置に従い、File をuploadします.
    1. plugins/　under に　userboxunder をupload
    2. admin/plugins/userbox/under に　adminunder のFile をmove
    3. public_html/userbox/under に　public_htmlunder のFile をmove
  3. Rootuser としてLogged in し，Plugin admin's Display でupdate を実行します

基本的なtesting version のupgrade procedure

Uninstall 、after delete, 、re Install してください.

Uninstall する前にConfiguration Settingのbackupを 作成しておくと、Install しなおした際にリストアすることができます.

Uninstall すると Database上の専用Data は、すべてdelete されます.

Uninstall すると uploadしたFile は、すべてdelete されます.

Table の追加変更が必要な case があります.

Group およびaccess rights の追加変更が必要な case があります.
