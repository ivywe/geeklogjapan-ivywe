概要

User 用の項目を自由に追加するPlugin です.
Install 時は、（userbox.user access rights がある）rootAdministratoronly 管理対象になります
Userがuserbox.user access rights があるGroup にregist されると、管理対象になります
Configuration Settingの「Logged in user をregist する」を「Yes」に変更すると、access rights regist にかかわりなく、すべてのLogged in user が管理対象になります

機能

  • Data 毎にfeaturedimageのregist がable：Additional field で、At the first time display されるImages （File save ）Type 項目
  • Images (featuredimage)付Category 別feed 発行
  • ogpMeta tag 編集（Data display 、Category 別Number list 、Category 別list ）
  • custom Setting：各site で独自のlanguage File 、Theme Settingがable
  • Category 別にTheme Settingがable
  • user にマイGroup のSettingをpermit することがable
    • マイGroup とは、Group name が_（アンダーバー）で始まるセキュリティグGroup （UserBox独自仕様）
    • Configuration Settingのuser にマイGroup のupdate をpermit するがYesになっている case 、or userbox.joingroupaccess rights がある case

関連Plugin

  • jQuery 1.3以上のVersion：jQeryPlugin が有効な case にonly date pickerが動作します.
    jQueryのロード順はUserBoxより前にしてください.    Gl2.0未満ではdate pickerは動作しません
  • XMLSitemapPlugin のConfiguration Settingのsite マップの内容に userbox をregist すると sitemap.xmlに出力します

制限事項

  1. Gl1.8未満のGeeklogではInstall できません.
  2. MySQL 5.0 未満ではInstall できません.
  3. GeeklogInstall 時に「UTF-8using」をcheck したsite only 対応しています.
  4. MySQLonly 対応しています.
  5. 主にXrea仕様 および SaaSes サーバで動作作検証しています.

注意

同一site にDataBoxがある case 、Version（幹番）は同じである必要があります.
